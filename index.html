신호는 5분 이내 갱신된 가격에서만 산출. 데이터 없는 종목은 자동 "대기"로 표시.

이번 주 주목 (Top Pick)

현재 시장·필터 풀에서 ● 지금 검토 신호 종목 중 우선순위 상위 3개를 페이지 상단에 표시.

우선순위 점수=(70 − 도달률) × 0.7
+ 시즌 진입월=+25점
+ 진입월 임박=+18점 (1~2개월 안에 시즌 시작 — 선취매)
+ 피크 시즌=+12점
+ 양수 등락률=+ min(등락률 × 2, 10)점

시장(국장/미장)·투자 스타일·필터를 바꾸면 Top Pick도 자동 갱신됩니다. 카드 클릭 시 해당 종목 행으로 스크롤.

종목 성격

종목마다 5가지 중 1~2개 부여. 같은 테마 안에서도 성격을 섞어 위험을 분산합니다.

주도주
테마에서 시총·매출 상위. 흐름이 가장 빠르게 반영되어 신호 확인용으로 적합.
모멘텀
시즌·이벤트 직전 가격이 빠르게 움직이는 종목. 진입·청산 타이밍이 핵심.
소형주
시총 작아 상승·하락 모두 큼. 한 종목에 큰 비중을 싣지 않는 것이 원칙.
회복 기대
부진했던 기업이 실적 회복 신호를 보이는 구간. 변동성 크니 실적 발표 전후 주의.
배당주
배당이 높고 가격 변동이 작음. 하락장에서 손실을 줄이는 역할.

투자 스타일

종목 필터로 사용. 한 종목이 안정·공격에 모두 속할 수 있습니다 (예: 삼성전자).

안정

공격
중소형주·모멘텀·소형주·회복 기대 종목 중심. 단기 매매와 고위험 고수익을 노릴 때.
전체

목표 도달률 · 상승 여력

현재가와 사전 설정한 목표가(과거 고점 또는 시즌 평균 피크가) 사이 비율로 계산합니다.

목표 도달률=현재가 / 목표가 × 100%
상승 여력=(목표가 - 현재가) / 현재가 × 100%

목표가는 과거 흐름을 보고 정성적으로 정한 값. 백테스트(과거 데이터 검증)와는 별개입니다.

매수·고점 구간 기준

매수 구간목표 도달률 < 70%
고점 근접목표 도달률 70 ~ 95%
과열목표 도달률 ≥ 95%

통계는 5분 이내 갱신된 최신 가격 종목만 포함합니다. 오래된 가격 데이터는 잘못된 신호를 줄 수 있어 제외.

백테스트 지표

별점(정성 평가)과 별개로, 과거 8년 데이터를 기준으로 한 검증 결과. 테마 헤더 옆 배지로 표시.

CAGR연평균 복리 수익률
Sharpe위험 대비 수익 (높을수록 좋음, 1.0 이상 양호)
MDD최대 낙폭 (낮을수록 좋음, -30% 이상은 주의)
승률연도별 + 수익 비율
vs 시장전략 CAGR - 시장 지수 CAGR (KOSPI / QQQ)

검증 기간 2018-01-01 ~ 2025-12-31 (8년). 시즌 전략은 매수월~매도월만 보유, 섹터 전략은 균등 가중 상시 보유.

테마 분석 데이터 (자동 산출)

테마 분석 페이지의 모든 지표는 종목 분석 페이지의 실시간 가격에서 자동 산출됩니다 (캘린더의 12개 산업 섹터 기준).

strength=(100 − 평균도달률) × 0.7 + 평균등락률 × 5 + 30
change=테마 종목들의 평균 등락률
leaders=등락률 상위 7개 종목
rank=strength 내림차순 자동 부여
RRG 좌표=(strength, change) 시계열 누적

5분마다 자동 갱신. 시장(국장/미장) 토글 시 즉시 재산출.

✦ 서버 추가 시 개선: 맥 스튜디오 서버를 두면 사용자 방문과 무관하게 매일 자동 저장 → 즉시 풀 궤적 가능. 또한 거래대금·시총 등 추가 데이터로 strength 정확도 향상.
